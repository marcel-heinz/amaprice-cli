# Security Policy

## Supported Versions

Security fixes are applied to the latest released version of `amaprice`.

## Reporting a Vulnerability

Please report vulnerabilities responsibly and do not disclose them publicly before a fix is available.

Preferred reporting methods:

1. GitHub Security Advisories (private): https://github.com/marcel-heinz/amaprice-cli/security/advisories/new
2. Direct email: marcel.heinz@gmail.com

When reporting, include:

- A clear description of the issue
- Steps to reproduce (proof of concept if possible)
- Potential impact
- Any suggested remediation

You can expect an initial response within 5 business days.
